# MAXIMUS RC TO BLUETOOH

Converting a common RC to use your cell phone as a controller...

## What will you need

- 7v or 8v motor;
- Servo motor
- Arduino Uno
- Bluetooth board H05
- 7v Battery
- Cell phone

## Environment setup

 - Install the code on arduino.
 - Download the app "Bluetooth RC Controller"
